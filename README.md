# paintfly
docs for paintfly

## 场景设计
### 后台新增资源（及逻辑）
1. 登陆ishabella，查看所有资源组（不同的应用服务）
2. 定义drd（即动态资源声明）
3. 为drd编写故事(命运之书)
4. 或者加载三方的命运之书
5. ishabella 测试调用

### 资源版本管理
1. 依赖资源自动提醒，测试
2. 回滚能力
3. 版本间数据记录 , 便于回滚数据处理

### 原生资源schema变更情况
1. light 每次起来时，自动检查schema变动
2. 变动提醒到ishabella

### 命运之书无法描述时
1. 利用命运之书的扩展能力扩展
2. 原则上统一扩展升级light实现功能
